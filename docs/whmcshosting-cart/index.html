Installation

It is very obvious that you need to install our WHMCS Hosting Cart (order form) template to use it. Don't worry, it's very simple to install and use. You can copy our (whmcs hosting cart) order form template directory and use an FTP program to upload the directory (rs_whmcshosting_cart) into your web hosting file manager. 

 Navigate to  WHMCS Root (installation) Directory > templates > orderforms
After upload completion, logged into your WHMCS Admin Area and 
 Navigate to Setup > General Settings
Select the Ordering tab
Select our (RS Whmcs Hosting Cart) order form template from the Default Order Form Template section.
Finally click Save Changes.
For more information please go to the below WHMCS doc website url 

				Change the default Order Form Template and/or Change for a specific product group.
				Learn what is Product Groups
That's it. Installation Complete.
